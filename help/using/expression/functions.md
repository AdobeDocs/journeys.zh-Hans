# 函数 {#concept_p1r_qj5_dgb}

一个函数可以具有不同的签名（一组不同的有序参数）。 函数签名可以具有0-N个表达式作为有序参数。

`<function name>`(`<expression as param 1>`, `<expression as param 2>`, ... ,`<expression as param N>`)

每个函数都有一个特定的返回类型。
