# toTimeZone {#toTimeZone}

将字符串值转换为时区。

## 类别

转化

## 函数语法

`toTimeZone(<parameter>)`

## 参数

| 参数 | 描述 |
|--- |--- |
| 字符串 | 字符串值必须包含区域ID。 它可以是字段引用或表达式 |

## 签名和返回的类型

`toTimeZone(<string>)`

返回时区。

## 示例

`toTimeZone("UTC")`

返回UTC。
