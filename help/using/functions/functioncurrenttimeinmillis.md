# currentTimeInMillis {#currentTimeInMillis}

返回当前时间（以新纪元毫秒为单位）。

## 类别

日期

## 函数语法

`currentTimeInMillis()`

## 参数

此函数不使用参数。

## 签名和返回类型

`currentTimeInMillis()`

返回整数。

## 示例

`currentTimeInMillis()`

返回“1544712617131”。
