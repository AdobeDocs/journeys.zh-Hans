# round {#round}

返回与参数最接近的整数值，并将整数舍入为正无穷大。

## 类别

数学

## 函数语法

`round(<parameters>)`

## 参数

* 小数
* 整数

## 签名和返回类型

`round(<decimal>)`

`round(<integer>)`

返回整数。

## 示例

`round(3.14)`

返回3。

`round(3.54)`

返回4。

`round(-3.14)`

返回–3。

`round(3)`

返回3。
