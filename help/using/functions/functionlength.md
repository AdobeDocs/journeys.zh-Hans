# 长度 {#length}

返回参数中字符串表达式的字符数。

## 类别

字符串

## 函数语法

`length(<parameters>)`

## 参数

* 字符串

## 签名和返回类型

`length(<string>)`

返回整数。

## 示例

`length("Hello World")`

返回11。
