# isEmpty {#isEmpty}

如果引數中的字串沒有字元，則傳回true。

## 类别

字符串

## 函式語法

`isEmpty(<parameters>)`

## 参数

* 字符串

## 簽章和傳回的型別

`isEmpty(<string>)`

傳回布林值。

## 示例

`isEmpty("")`

傳回true。

`isEmpty("Hello World")`

傳回false。
