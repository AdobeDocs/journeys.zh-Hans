# countOnlyNull {#countOnlyNull}

计算列表中的null值数。

## 类别

聚合

## 函数语法

`countOnlyNull(<listAny>)`

## 参数

| 参数 | 类型 |
|-----------|------------------|
| 列表 | listString |
| 列表 | listBoolean |
| 列表 | listInteger |
| 列表 | listDecimal |
| 列表 | listDuration |
| 列表 | listDateTime |
| 列表 | listDateTimeOnly |

## 签名和返回类型

`countOnlyNull(<listAny>)`

返回整数。

## 示例

`count([10,2,10,null])`

返回1。
