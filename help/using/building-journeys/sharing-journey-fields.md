# 历程字段 {#sharing-journey-fields}

此Mixin用於歷程結構描述（與journeyStepEvent相關）。 它包含下列欄位：

## journeyID

主要歷程的ID。

类型：字符串

## journeyVersionID

歷程版本的ID。 此ID代表歷程的身分。

类型：字符串

## name

歷程的名稱。

类型：字符串

## 描述

歷程的說明。

类型：字符串

## version

版本，表示為 `major`.`minor`

类型：字符串
