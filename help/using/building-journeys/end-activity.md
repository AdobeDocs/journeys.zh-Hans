# 结束活动{#section_vqp_4ft_dgb}

此 **[!UICONTROL End]** 利用活动，可标记历程每个路径的结尾。 这并非强制性要求，但我们推荐使用以便能够清晰了解情况。实际上，如果历程有多个结束活动，我们建议您为每个结束活动添加标签，以使报告更易于阅读。请参阅[此页](../reporting/about-journey-reports.md)。
